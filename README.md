## 下载

### Docker

```
docker pull apachecn0/huggingface-doc-zh
docker run -tid -p <port>:80 apachecn0/huggingface-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g huggingface-doc-zh
huggingface-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
